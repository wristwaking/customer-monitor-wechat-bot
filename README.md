# wechat-customer-monitor 微信客服数据监控插件程序
wechat-customer-monitor

# 启动程序扫码登录微信
```bash
node bot.js
```

# 统计指定日期范围客服数据
```
node tool.js

填写需要统计客服数据的日期: YYYY.MM.DD （例如：2025.01.01）
```
